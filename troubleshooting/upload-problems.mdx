# Upload Problems

Resolve common upload issues quickly and get your content into Kernel MKII. This guide covers file upload problems, URL import issues, and processing errors.

![Upload Troubleshooting](/images/upload-troubleshooting-overview.png)

## Common Upload Errors

### File Upload Errors

#### **"File too large" Error**

**Error Message**: "File exceeds maximum upload size"

**Causes**:
- File larger than 50MB limit
- Multiple large files selected at once
- Network upload limits

**Solutions**:
1. **Reduce File Size**
   - Compress PDF files using online tools
   - Reduce image resolution
   - Split large documents into sections
   - Use file compression software

2. **Upload Files Individually**
   - Upload one large file at a time
   - Avoid batch uploading multiple large files
   - Monitor upload progress

3. **Contact Administrator**
   - Request increased file size limits
   - Discuss enterprise upload options
   - Get help with large file workflows

![File Size Error](/images/file-size-error.png)

#### **"Unsupported file format" Error**

**Error Message**: "File type not supported"

**Supported Formats**:
- **Documents**: PDF, DOC, DOCX, TXT, RTF
- **Presentations**: PPT, PPTX
- **Spreadsheets**: XLS, XLSX, CSV
- **Images**: JPG, PNG, GIF, BMP
- **Web**: HTML files

**Solutions**:
1. **Convert File Format**
   - Save Word documents as PDF
   - Export presentations to PDF
   - Convert images to JPG or PNG
   - Use online file converters

2. **Check File Extension**
   - Ensure file has correct extension
   - Rename file with proper extension
   - Verify file isn't corrupted

3. **Alternative Upload Methods**
   - Use URL import for web content
   - Copy text content to supported format
   - Screenshot images for visual content

#### **"Upload failed" Error**

**Error Message**: "Upload could not be completed"

**Common Causes**:
- Network connectivity issues
- Browser problems
- Server temporary issues
- File corruption

**Solutions**:
1. **Check Network Connection**
   - Verify internet connectivity
   - Try different network if available
   - Restart router/modem if needed

2. **Browser Troubleshooting**
   - Refresh the page and try again
   - Clear browser cache and cookies
   - Try different browser
   - Disable browser extensions temporarily

3. **File Issues**
   - Try uploading different file
   - Check if file opens normally
   - Scan file for corruption
   - Try uploading from different location

![Upload Failed Error](/images/upload-failed-error.png)

### Processing Errors

#### **"Processing timeout" Error**

**Error Message**: "File processing took too long"

**Causes**:
- Very large files
- Complex document structure
- High server load
- Network issues during processing

**Solutions**:
1. **Wait and Retry**
   - Processing may complete after delay
   - Check file status after 10-15 minutes
   - Retry upload if status remains "Processing"

2. **Optimize File**
   - Reduce file size if possible
   - Simplify complex documents
   - Remove embedded media if not needed

3. **Upload During Off-Peak Hours**
   - Try uploading during less busy times
   - Early morning or late evening often better
   - Avoid peak business hours

#### **"Text extraction failed" Error**

**Error Message**: "Could not extract text from file"

**Causes**:
- Scanned documents without OCR
- Password-protected files
- Corrupted files
- Complex layouts

**Solutions**:
1. **OCR Processing**
   - Use OCR software to extract text first
   - Upload text-searchable PDF version
   - Try online OCR tools

2. **Remove Protection**
   - Remove password protection if possible
   - Save copy without security settings
   - Contact document owner for unprotected version

3. **Alternative Formats**
   - Convert to different format
   - Manually type key content
   - Upload summary document

![Processing Error](/images/processing-error.png)

### URL Import Problems

#### **"URL not accessible" Error**

**Error Message**: "Could not access the provided URL"

**Causes**:
- URL requires login
- Page behind paywall
- Broken or expired link
- Network restrictions

**Solutions**:
1. **Verify URL**
   - Check URL works in browser
   - Remove tracking parameters
   - Try shortened URL if available
   - Ensure URL is publicly accessible

2. **Alternative Access**
   - Save webpage as PDF first
   - Copy content to document
   - Use web archive version
   - Contact content owner

3. **Login Requirements**
   - Access page in browser first
   - Copy content manually if needed
   - Use alternative public sources

#### **"Content extraction failed" Error**

**Error Message**: "Could not extract content from webpage"

**Causes**:
- JavaScript-heavy websites
- Complex page layouts
- Anti-scraping protection
- Dynamic content loading

**Solutions**:
1. **Try Alternative URLs**
   - Use print-friendly page version
   - Try mobile version of page
   - Look for PDF version on site

2. **Manual Content Copy**
   - Copy text content manually
   - Paste into document and upload
   - Save webpage as PDF

3. **Contact Support**
   - Report problematic websites
   - Request enhanced extraction support
   - Provide feedback on failed URLs

![URL Import Error](/images/url-import-error.png)

## Upload Performance Issues

### Slow Upload Speeds

#### **Identifying Slow Uploads**

**Symptoms**:
- Upload progress bar moves very slowly
- Files take unusually long to upload
- Frequent upload timeouts

**Causes**:
- Slow internet connection
- Large file sizes
- Network congestion
- Browser issues

**Solutions**:
1. **Check Connection Speed**
   - Test internet speed using speed test tools
   - Compare with expected speeds
   - Contact ISP if speeds are low

2. **Optimize Upload Process**
   - Upload during off-peak hours
   - Close other bandwidth-heavy applications
   - Use wired connection instead of WiFi
   - Upload files one at a time

3. **Browser Optimization**
   - Close unnecessary browser tabs
   - Disable browser extensions
   - Clear browser cache
   - Use recommended browsers (Chrome, Firefox, Safari)

### Multiple Upload Failures

#### **Batch Upload Issues**

**Symptoms**:
- Some files in batch upload fail
- Inconsistent upload success
- Random upload errors

**Solutions**:
1. **Upload Files Individually**
   - Upload one file at a time
   - Identify which files cause problems
   - Process successful uploads first

2. **Check File Consistency**
   - Verify all files are similar format
   - Check for corrupted files
   - Remove problematic files from batch

3. **System Resource Issues**
   - Close other applications
   - Restart browser
   - Check available disk space

## Browser-Specific Issues

### Chrome Issues

**Common Problems**:
- Upload stuck at 100%
- "Aw, Snap!" page crashes
- Memory issues with large files

**Solutions**:
1. **Clear Chrome Data**
   - Clear browsing data
   - Reset Chrome settings
   - Disable problematic extensions

2. **Memory Management**
   - Close other Chrome tabs
   - Restart Chrome
   - Check available RAM

### Firefox Issues

**Common Problems**:
- Upload progress not showing
- Firefox becomes unresponsive
- File drag-and-drop not working

**Solutions**:
1. **Firefox Troubleshooting**
   - Refresh Firefox
   - Check for updates
   - Disable add-ons temporarily

2. **Upload Method**
   - Use file selection instead of drag-and-drop
   - Try private browsing mode
   - Check Firefox security settings

### Safari Issues

**Common Problems**:
- Upload interruptions
- File type recognition issues
- Progress tracking problems

**Solutions**:
1. **Safari Settings**
   - Check Safari security settings
   - Enable JavaScript
   - Clear Safari cache

2. **macOS Permissions**
   - Check file access permissions
   - Verify Safari has necessary permissions
   - Try uploading from different folder

## Mobile Upload Issues

### Mobile Browser Problems

**Common Issues**:
- File selection not working
- Upload progress not visible
- App crashes during upload

**Solutions**:
1. **Use Mobile App**
   - Download official mobile app if available
   - Mobile apps often handle uploads better
   - Better progress tracking and error handling

2. **Browser Alternatives**
   - Try different mobile browser
   - Use desktop mode in mobile browser
   - Clear mobile browser cache

### File Access Issues

**Problems**:
- Can't access files on mobile device
- Files not visible in file picker
- Permission denied errors

**Solutions**:
1. **Check Permissions**
   - Grant file access permissions to browser
   - Check app permissions in device settings
   - Allow access to photos/documents

2. **File Location**
   - Move files to easily accessible location
   - Save files to Downloads folder
   - Use cloud storage integration

## Prevention and Best Practices

### Pre-Upload Checklist

1. **File Preparation**
   - ✅ File under 50MB size limit
   - ✅ Supported file format
   - ✅ File opens correctly
   - ✅ No password protection

2. **Network Preparation**
   - ✅ Stable internet connection
   - ✅ Sufficient bandwidth available
   - ✅ No VPN issues
   - ✅ Firewall allows uploads

3. **Browser Preparation**
   - ✅ Updated browser version
   - ✅ JavaScript enabled
   - ✅ Cookies enabled
   - ✅ Sufficient browser memory

### Upload Optimization Tips

#### **For Large Files**
1. **Compress Before Upload**
   - Use PDF compression tools
   - Optimize images
   - Remove unnecessary embedded content

2. **Upload Strategy**
   - Upload during off-peak hours
   - Use stable, fast internet connection
   - Avoid uploading multiple large files simultaneously

#### **For Batch Uploads**
1. **Organize Files**
   - Group similar file types
   - Check all files beforehand
   - Remove any problematic files

2. **Monitor Progress**
   - Watch upload progress closely
   - Be ready to retry failed uploads
   - Take note of successful uploads

## When to Contact Support

### Escalation Criteria

**Contact Support When**:
- Multiple upload methods fail
- Error persists after troubleshooting
- System-wide upload issues
- Enterprise-specific problems

### Information to Provide

1. **Error Details**
   - Exact error message
   - Screenshot of error
   - Time when error occurred
   - Browser and version

2. **File Information**
   - File type and size
   - File name (if relevant)
   - Source of file
   - Upload method used

3. **System Information**
   - Operating system
   - Browser and version
   - Network connection type
   - Any recent system changes
